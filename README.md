# README
Simulates tensile tests of nanowires using Matplotlib.

### Available simulation parameters

max_force = 2
time_step = 0.02
lattice_width = 4
lattice_depth = 4
lattice_height = 8
simulation_length = 3 # seconds
force_x_component = 1
force_y_component = 1
force_z_component = 0

### Supported crystal structures
* Simple cubic
* Body-centered cubic
* Face-centered cubic
* Hexagonal close packed
